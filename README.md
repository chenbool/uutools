# uutools - 魔兽世界小号交易拦截插件

为魔兽世界添加的阻止小号交易插件，防止机器人骚扰和金币诈骗。

## 特性

- 阻止 1 级小号交易
- 防止机器人骚扰
- 拦截脚本强制交易
- 防止金币诈骗
- 支持 SHADOWLANDS 和 WOW CLASSIC
- 不干扰当前热键设置或配置

## 支持平台

- 国服
- 台服
- 港服

## 使用说明

该插件为静默运行插件，安装后自动生效，无需额外配置。

### 功能说明

- **阻止小号交易**：自动拦截 1 级角色的交易请求
- **防骚扰**：防止机器人自动发送的交易请求
- **防骗金**：拦截脚本强制交易，防止金币诈骗
